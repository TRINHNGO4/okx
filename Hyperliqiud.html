📋 Hướng dẫn cài đặt Google Sheets

  1. Mở Google Sheets → Tạo sheet mới
  2. Vào Extensions → Apps Script
  3. Xóa code cũ, dán code từ nút bên dưới
  4. Nhấn Deploy → New deployment → Web app
  5. Chọn Execute as: Me và Who has access: Anyone
  6. Copy URL → dán vào ô Apps Script URL trong Cài đặt
